## POKEDEX

Demo of NextJS' SSG and SSR with [chakra-ui](https://chakra-ui.com/)

**Requirements**

- node 16.\*
- yarn

**Installation**

- Clone repo
- Open terminal, cd to repo and run the command below

```
yarn
yarn dev
```

- Open browser and go to http://localhost:3000
