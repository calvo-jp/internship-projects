## POKEDEX

Demo of NextJS' SSG and SSR with chakra-ui and pokedex API

**Requirements**

- node 16.\*
- yarn

**Installation**

- Clone repo
- Go to repo and rename `.env.example` to `.env.local`
- Open terminal, cd to repo and run the command below

```
yarn
yarn dev
```

- Open browser and go to http://localhost:3000
